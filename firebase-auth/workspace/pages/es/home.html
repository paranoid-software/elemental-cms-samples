Ejemplo Firebase Auth

Integración de Autenticación con Firebase y Elemental CMS

Este ejemplo de Elemental CMS esta hecho con <3 utilizando:

  • jQuery
  • Plugster
  • Bulma, y
  • Google Firebase SDK

Intenta demostrar el uso de Firebase como mecanismo de autenticación y repositorio de usarios. Al mismo tiempo que ilustra sobre temas como:

  • Manejo de sesiones
  • Páginas multi-idioma
  • Uso de dependencias globales
  • Uso de dependencias locales (páginas y snippets)
  • Inclusión de snippets
  • Soporte multi-idioma en snippets
