Firebase Auth Sample

Elemental CMS Firebase Auth Integration

This Elemental CMS sample is made with <3 using:

  • jQuery
  • Plugster
  • Bulma, and
  • Google Firebase SDK

It tries to demonstrate the use of Firebase as users repository. And on its way also illustrates topics like:

  • Session managament
  • Multi language pages
  • Global dependencies usage
  • Local dependencies usage
  • Snippets inclusion
  • Snippets multi language support
